Suburb pages framework

Suburb page framework for Melbourne and Sydney

Use this structure to build consistent suburb pages while keeping each suburb genuinely useful and distinct.

Required sections

  • Suburb overview with city context and practical user guidance
  • Listings snapshot pathway from data/listings.json
  • Category shortcuts relevant to user intent
  • Booking-style shortcuts (SMS/email, phone, online, walk-in)
  • Owner claim/update call-to-action

Manual quality rules

  • Only include facts verified from official business websites
  • Do not copy external review text or star ratings
  • No guaranteed outcomes or medical claims
  • Use consistent field values from the data dictionary

Starter template and examples

Use suburb-template.html to create new suburb pages. Existing suburb pages can be used as structure references only, with facts verified individually.
